Welcome to {{appname}}!

We recommend setting aside some time to explore the docs and get your team set up. If you ever need additional help, our community forums are full of useful tips.

To follow a repository that's already set up on {{appname}}:
  1. Log in to {{appname}} and click 'Add Projects' in the left sidebar.
  2. Click 'Follow Project' next to the project you'd like to follow.

If you aren't seeing the project you'd like to follow, check that you are working in the correct VCS organization using the org selection dropdown in the top lefthand corner of the page.

To set up a new repository on {{appname}}:
  1. In the {{appname}} UI, click 'Add Projects' in the lefthand sidebar and then click 'Set up project' next to the repository you want to set up.
  2. Select the language for the project from the drop-down to generate a config.yml file, and make any edits necessary in the text editor below.
  3. Click "Start Building" and then "Add Config"

Feel free to check out our getting started guides for Linux, Windows, and macOS for information specific to each build environment.
